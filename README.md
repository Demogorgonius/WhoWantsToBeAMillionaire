**Who wants to be a Millionaire**

*Stack:* UIKit \ SnapKit
Возможность выиграть 1000000 RUB, ответив на 15 вопросов

*Особенности:*
- 2 несгораемые суммы (1000 RUB / 32000 RUB)
- Возможность забрать деньги
- Результаты игры
- Подсказки

*Подсказки:*
- 50/50
- Звонок другу
- Помощь зала
- Право на ошибку
